FPGA Project

  • Load optimization system with FPGA

On the project there are two dc motor. One of them driven with external unknown signal. FPGA recognizes that signal with output of that motor(quadrature signals). Then optimizes other motor according to load's rpm. Also FPGA sends current rpm with UART to computer. So, we can see the situation real time.
